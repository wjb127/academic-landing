■ 각 구현 사항별 테스트 방법

【1】 메인 페이지 텍스트 수정 확인
□ 테스트 방법:
  1. 홈페이지 최상단 확인
  2. "제1기 원우 모집" 텍스트 표시 확인 (이전: 2025년 제1기 모집)
  3. "부산 힐튼호텔 5층" 표시 확인 (이전: 부산 그랜드볼룸)

□ 확인 위치: 메인 화면 상단 Hero 섹션

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【2】 파트너사 로고 이미지 확인
□ 테스트 방법:
  1. 페이지 스크롤하여 "아카데미 소개" 섹션 이동
  2. 부산일보, 비온미디어, 쟁글 로고 이미지 확인
  3. 로고가 텍스트가 아닌 실제 이미지로 표시되는지 확인

□ 확인 위치: 아카데미 소개 섹션 상단

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【3】 커리큘럼 정보 확인
□ 테스트 방법:
  1. "커리큘럼" 섹션으로 스크롤
  2. 1주차 클릭 → 심준식 대표 오리엔테이션 확인
  3. 8주차 클릭 → 차란차 본부장, 도이치모터스그룹 확인

□ 확인 위치: 커리큘럼 섹션 (주차별 클릭)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【4】 지원 방법 UI 테스트
□ 테스트 방법:
  1. 메인 화면 "지금 지원하기" 녹색 버튼 클릭
  2. 모달창 팝업 확인
  3. 두 가지 옵션 표시 확인:
     - 온라인 지원서 작성 (녹색 박스)
     - 입학원서 다운로드 (파란색 박스)

□ 온라인 지원 테스트:
  1. "온라인 지원서 작성" 클릭
  2. Google Docs 페이지 새 탭 열림 확인
  3. 템플릿 복사 페이지 표시 확인

□ HWP 다운로드 테스트:
  1. "입학원서 다운로드" 클릭
  2. HWP 파일 다운로드 시작 확인
  3. 파일명: "부산 디지털금융·블록체인 아카데미 안내문 및 입학원서.hwp"

【5】 지원 접수 정보 확인
□ 테스트 방법:
  1. 페이지 하단 "지원 방법" 섹션 이동
  2. 제출 서류 확인:
     - 증명사진만 표시 (입학원서 항목 없음)
     - "원서 부분 혹은 사진파일로 첨부" 문구 확인
  3. 접수 방법 정보 확인:
     - 이메일: jswon@busan.com
     - 방문/우편: 부산광역시 동구 중앙대로 365, 부산일보 전략기획부(5층)

□ 확인 위치: 지원 방법 섹션

【7】 특별 혜택 내용 확인
□ 테스트 방법:
  1. "아카데미 소개" 섹션 이동
  2. 지원 자격 아래 노란색 박스 확인
  3. 특별 혜택 내용 확인:

■ 모바일 반응형 테스트

□ 테스트 방법:
  1. 브라우저 개발자 도구 열기 (F12)
  2. 반응형 디자인 모드 전환 (Ctrl+Shift+M)
  3. 다양한 디바이스 선택:
     - iPhone SE (375px)
     - iPhone 12 Pro (390px)
     - iPad (768px)
  4. 각 화면에서 레이아웃 깨짐 없는지 확인

□ 주요 확인 사항:
  - 텍스트 가독성
  - 버튼 클릭 가능 여부
  - 이미지 비율 유지
  - 모달창 정상 표시

■ 문제 발생 시 확인 방법

□ 콘솔 에러 확인:
  1. F12 → Console 탭
  2. 빨간색 에러 메시지 확인
  3. 에러 내용 캡처

□ 네트워크 확인:
  1. F12 → Network 탭
  2. 페이지 새로고침 (F5)
  3. 404 에러 또는 Failed 항목 확인
